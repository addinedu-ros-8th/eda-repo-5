## 개요
1. 반려동물을 키우는 인구가 가파르게 증가하는 요즘 끝까지 책임지지 못하고 유기하는 문제가 사회적 문제로 떠오르고 있습니다. 이를 막고자 반려동물 등록제를 시행하고 있지만 아직 많은 사람들이 등록을 않하는 실정입니다. 여기서 저희는 반려동물 등록건수와 유기동물건수를 분석하여 등록건수 증가율에 따른 유기율을 알아볼 것입니다.
